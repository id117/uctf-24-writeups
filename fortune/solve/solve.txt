1. функция exec позволяет выполнить вводимый код (python3)
2. from random import * импортирует все содержимое модуля random
3. все программные датчики случайных чисел должны быть воспроизводимыми 
    (последовательность генерируемых случайных чисел должна зависеть от инициализирующего вектора)
4. функция seed() модуля random позволяет задать инициализирующий вектора
5. как это работает:
```python3
>>> from random import *
>>> seed(0)
>>> random()
0.8444218515250481
>>> seed(0)
>>> random()
0.8444218515250481
```
6. мы знаем, что число округляется до трех знаков после запятой
7. float в python3 можно записывать без целой части:
```python3
>>> .844 == 0.844
True
```
Таким образом, решение может быть, например, такое:
.844;seed(0)
или, если не знать про пункт 7, то, например, такое (придется подбирать подходящий seed):
0.2;seed(85)
